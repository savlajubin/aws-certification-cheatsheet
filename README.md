# AWS Certification Cheat Sheets

One-page interactive study guides to help you pass AWS certification exams. Open in a browser, read through once, and you're ready.

## Cheat Sheets

| Exam | Code | Link |
|------|------|------|
| AWS Cloud Practitioner | CLF-C02 | [Open](aws-ccp-cheatsheet.html) |
| AWS AI Practitioner | AIF-C01 | [Open](aws-aif-cheatsheet.html) |

## Features

- Covers every exam domain with correct weightings
- Service tables, visual diagrams, and decision trees
- Exam tips, keyword triggers, and common traps
- Searchable — type anything in the search bar
- Light/dark theme with system preference detection
- Sidebar navigation with scroll tracking
- Reading progress bar
- Zero dependencies — just static HTML, CSS, and JS

## Usage

Open `index.html` in a browser, or visit the GitHub Pages site:

```
https://<username>.github.io/<repo-name>/
```

No build step, no framework, no install.

## Structure

```
index.html                 Landing page
```
